How Do I Use if case let Syntax in Swift?

if case let is just an abbreviated form of switch:

The pattern to match against goes in exactly the same place, after the case let keywords. But the order looks reversed, as the expression has moved right, after the = operator.

if case let Puppy.mastiff(droolRating, weight) = fido {
... // tell fido he's a good boy ...

is equal to

switch fido {
case let Puppy.mastiff(droolRating, weight):
... // tell fido he's a good boy ...

let binds associated values

Putting let after case means you can omit it for all subsequent associated value bindings. Or you can write it before each value:

if case Puppy.mastiff(let droolRating, let weight) = fido {
... // does that pup need an extra treat? ...

Bind a tuple with all the values at once:

if case let Puppy.mastiff(characteristics) = fido {
... // let's check this one out ...

Omit let entirely if you don't need the associated values:

if case Puppy.mastiff = fido {
... // just hand me the pupster ...

Add where clauses using commas

if case let Puppy.mastiff(droolRating, _) = fido, droolRating > 4 {
... // you're going to need a new couch ...

Add where clauses in guard statements

guard case let Puppy.mastiff(droolRating, _) = fido, droolRating > 10 {
... // that dog is not coming in the house ...

Iterations with for statements and where clauses

for case let Puppy.mastiff(droolRating, weight) in rescuePups where droolRating > 5 && weight > 15.0 {
... // I'll take the chubby, soggy ones please ...
